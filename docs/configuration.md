# Configuration

Config is stored as JSON under LocalAppData.

You can:
- Edit it via the UI (**Save** button)
- Or print it / change it with the CLI

Show current config:

```powershell
pom config
```

Set values:

```powershell
pom config set work 30
pom config set break 5
pom config set long 15
pom config set cycles 4
pom config set auto true
pom config set popup true
pom config set sound true
```
