# Tray + Window UI

## Window

The window shows:
- Current phase (Work/Break/LongBreak) and cycle count
- Countdown
- Buttons: Start / Pause / Resume / Stop
- Settings for the next start (durations + toggles)

## Tray

- Left-click tray icon: opens the window
- Right-click tray icon: context menu

Context menu:
- Open
- Start / Start (Force)
- Pause / Resume / Stop
- Status
- Exit
